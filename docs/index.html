CommentDetails

Post a new comment

Authorizations:
bearerAuth
Request Body schema: application/json
required
username
required
string [ 3 .. 100 ] characters

The commenter's username

comment
required
string [ 3 .. 250 ] characters

The contents of the comment

imageId
required
string [ 3 .. 50 ] characters

The image ID identifying where the comment was posted

Responses

Local server

http://localhost:3000/api/v1/comment

Request samples

  • Payload
Content type
application/json
{
  • "username": "string",
  • "comment": "string",
  • "imageId": "string"
}

Response samples

  • 201
Content type
application/json
{
  • "id": "string",
  • "username": "string",
  • "comment": "string",

}
